# Security Policy

## Scope

This policy covers all actively maintained repositories in this organisation.

## Reporting

**Please do not open a public GitHub issue for security concerns.**

Email **anirudh@wedogood.in** with the subject line `[SECURITY]` and include:

- A description of the issue
- Steps to reproduce it
- The potential impact
- Any suggested mitigations (optional)

We will acknowledge your report within **48 hours** and aim to complete triage within **5 business days**.

## Disclosure

We ask that you give us a reasonable period to address the issue before any public disclosure. We will credit researchers in release notes unless you prefer to remain anonymous.
